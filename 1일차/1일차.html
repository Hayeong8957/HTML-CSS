나의 첫 html파일 만들기

1일차 실습에 사용된 태그 종류 설명

  • !DOCTYPE html : 이 웹페이지가 HTML로서 만들어졌다는 것을 표현하기 위해서 문서의 시작에 추가하는 코드이다.
  • html : body태그와 head태그를 감싸는 태그
  • body / head : 본문은 body태그로, 본문을 설명하는 태그는 head태그를 사용한다.
  • heading : 제목태그, h1 ~ h6, 숫자가 올라갈수록 글자크기가 작아진다.
  • p : paragraph, 즉 문단의 약자로, 하나의 문단을 만들 때 쓰인다.
  • li : 목차(list) 태그이다.
  • ul : 목록을 다른 목록과 구분할 수 있도록 경계를 지어주는 태그이다.
  • strong : 글자 강조 태그이다.
  • br : 줄바꿈 태그이다.
  • img : 이미지를 삽입하게 해주는 태그인데, img 태그에 source의 줄임말인 src를 붙여서 이미지를 불러올 수 있게 한다.
